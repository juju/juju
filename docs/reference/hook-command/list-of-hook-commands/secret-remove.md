# `secret-remove`
## Summary
Remove an existing secret.

## Usage
``` secret-remove [options] <ID>```

### Options
| Flag | Default | Usage |
| --- | --- | --- |
| `--revision` | 0 | remove the specified revision |

## Examples

    secret-remove secret:9m4e2mr0ui3e8a215n4g


## Details

Remove a secret with the specified URI.
