# `secret-ids`
## Summary
Print secret IDs.

## Usage
``` secret-ids [options] ```

### Options
| Flag | Default | Usage |
| --- | --- | --- |
| `--format` | smart | Specify output format (json&#x7c;smart&#x7c;yaml) |
| `-o`, `--output` |  | Specify an output file |

## Examples

    secret-ids


## Details

Returns the secret ids for secrets owned by the application.
